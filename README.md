## About

Fast-codded snake game for terminal.

### Technologies

* Language: **Python**

## Installing

**Firstly** clone the project.

```git
git clone https://github.com/l1ve4code/terminal-snake.git
```

**Secondly** install necessary libraries

**Thirdly** run from terminal

```cmd
python main.py
```
